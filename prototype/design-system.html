Solace Design System

Space-optimized visual language for astronaut mental wellness

Informed by Meta FAIR's TRIBE v2 neuroscience research · Designed for low-light legibility, glove-friendly touch targets, and low-distraction interfaces

Design Principles (TRIBE v2)

Minimal Motion

Avoid taxing the motion network during cognitive-load screens. Subtle transitions only.

Icon-Forward

Reduce language network load with icon-based navigation. Minimize dense text.

Multi-Sensory

Pair nature sounds with visuals for restoration. Engage auditory + visual networks together.

Default Mode

Restoration features engage the default mode network through journaling and nature scenes.

Visual Clarity

High-contrast dark theme optimized for visual processing. Calming palette, no clutter.

Accessibility & Space Context Notes

Low-Light Legibility

Dark background (#060D19) with high-contrast text (#E2E8F0) exceeds WCAG AAA ratio of 15.2:1. All color pairs tested for 4.5:1 minimum.

Glove-Friendly Targets

Minimum 48px touch targets throughout. Tab bar icons at 72px width. Buttons have generous padding for EVA glove use.

Low-Distraction Interface

Minimal animation (TRIBE v2 motion principle). No auto-scrolling. Calm color palette. Icon-forward navigation reduces cognitive load.
