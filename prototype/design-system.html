Solace Design System

Space-optimized visual language for astronaut mental wellness.
Informed by Meta FAIR's TRIBE v2 neuroscience research.

Purpose-built for low-light legibility, glove-friendly touch targets, and low-distraction interfaces

Design Principles (TRIBE v2)

Auditory Network

Multi-Sensory Stimuli

Pair nature sounds with visuals for deeper cortical engagement. Audio + visual together activates more cortex than either alone.

Default Mode Network

Nature & Journaling

Engage the brain's rest-state network through nature exposure and reflective journaling. Designed for restoration, not stimulation.

Visual Network

Dark High-Contrast Theme

Optimized for visual processing without strain. Soft gradients over harsh borders. NASA blue palette reads well in low light.

Motion Network

Minimal Animation

Excessive motion taxes the brain. Only subtle, purposeful animation — the breathing ring, gentle transitions. Never decorative.

Language Network

Icon-Forward Navigation

Reduce language processing load. Large icons, visual selectors, minimal text. The interface should be scannable in seconds.

Accessibility & Space Context

Low-Light Legibility

Dark navigation (#070D1A) with high-contrast text (#E6EDF3). Light content area (#E9ECF2) maintains readability. Tested at 12-17px minimum body text sizes.

WCAG AA Contrast

Glove-Friendly Targets

Minimum 48px touch targets across all interactive elements. Tab bar buttons at 48x56px. Form controls at 44px minimum height. Generous padding on all tappable areas.

48px Minimum

Low-Distraction Interface

Minimal animation per TRIBE v2 Motion Network research. No decorative motion. Subtle transitions (0.15-0.3s ease). Breathing ring is the only sustained animation — purposeful, not decorative.
